# ✨ KI ENTERTAINMENT | 홈페이지 리디자인

신생 기획사 **KI ENTERTAINMENT**의 공식 홈페이지를 모던하고 세련된 디자인으로 완전히 리디자인한 프로젝트입니다.

🌐 **[라이브 데모](https://uiux0718.github.io/redesign-project/)** | 📱 **반응형 완벽 지원** | ♿ **접근성 우선**

---

## 🎯 프로젝트 개요

기존 KI ENTERTAINMENT 홈페이지를 분석하여, **모던한 다크테마**, **일관된 디자인 언어**, **완벽한 반응형 대응**을 중심으로 개선했습니다.

신생 기획사이면서 소속 아티스트는 **명곡 맛집 ON&OFF(ONF)** 만을 보유하고 있기 때문에, **ONF 중심의 깔끔한 멤버 그리드**로 통합하여 더욱 포커스된 사용자 경험을 제공합니다.

---

## 🎨 주요 특징

- ✨ **모던 다크테마** — 진한 배경 + 밝은 텍스트 + 신선한 그린 악센트
- 🎤 **ONF 중심 통합** — 온팀/오프팀 분리 제거, 6명 멤버를 한눈에
- 📱 **완벽한 반응형** — 3단계 최적화 (데스크톱 3컬럼 → 태블릿 2컬럼 → 모바일 1컬럼)
- ♿ **접근성 최우선** — Skip link, ARIA 레이블, 의미있는 시맨틱 HTML
- 🚀 **성능 최적화** — 이미지 lazy-loading, 폰트 preconnect
- ⚡ **정적이지만 우아한** — 외부 라이브러리 제로, 순수 HTML/CSS/JS

## 💡 리디자인 의사결정

### 1️⃣ ON/OFF 팀 통합
- 신생 기획사 (소속 아티스트 ONF만)
- 팀 구분이 불필요 → **단일 그리드 통합**

### 2️⃣ 멤버 카드 클릭 제거
- 프로필 페이지 부재
- **정적이지만 우아한 프레젠테이션** 추구

### 3️⃣ 다크테마 선택
- 음악 산업 트렌드
- 프리미엄한 이미지 표현
- 그린 악센트로 신선함 극대화
