---

 프로젝트 개요

기존 KI ENTERTAINMENT 홈페이지를 분석하여, 모던한 다크테마, 일관된 디자인 언어, 반응형 대응을 중심으로 개선했습니다.

현재 소속 아티스트는 보이그룹 ON&OFF(ONF) 보유하고 있기 때문에, ONF 중심의 깔끔한 멤버 그리드로 통합하여 더욱 포커스된 사용자 경험을 제공합니다.

---

 🎨 주요 특징

-  모던 다크테마 — 진한 배경 + 밝은 텍스트 + 신선한 그린 악센트
- ONF 중심 통합— 6명 멤버를 한눈에
- 완벽한 반응형— 3단계 최적화 (데스크톱 3컬럼 → 태블릿 2컬럼 → 모바일 1컬럼)
- 접근성 최우선 — Skip link, ARIA 레이블, 의미있는 시맨틱 HTML
- 성능 최적화 — 이미지 lazy-loading, 폰트 preconnect
-  정적이지만 우아한 — 외부 라이브러리 제로, 순수 HTML/CSS/JS

---


---

 🛠️ 기술 스택

- **HTML5** — 시맨틱 마크업, ARIA 역할
- **CSS3** — Flexbox, Grid, CSS 변수, `clamp()` 반응형
- **JavaScript** — 비디오 재생, 부드러운 스크롤
- **Pretendard** — 한글 웹폰트
